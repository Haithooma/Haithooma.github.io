Restaurant Ordering System - Requirements

Customer requirements:

  1. The system shall prompt the user to register providing the following information for contact:
    • Name.
    • Phone number.
    • Email.
    • Address, by filling the form or by using the embedded map.
  2. The system shall prompt the customer to choose the type of bread
    Types are:
    • White bread.
    • Brown bread.
  3. The system shall prompt the customer to choose the size of the sandwich
    Sizes are:
    • 6 inches.
    • 12 inches.
  4. The system shall allow the customer to choose one or more kind of vegetables.
  5. The system shall allow the customer to choose one or more types of cheese.
  6. The system shall allow the customer to choose one or more sauces.
  7. The system shall allow the customer to choose whether they want drinks or not, if they do, then the list of drinks shall appear for them to choose from.
  8. The system should allow the customer to choose a snack separately, and the customer can choose between the different provided snacks.
  9. The system should allow the customer to buy a drink separately, and provides a list of the available drinks and sizes.

Restaurant Employee:

  1. The system should allow the employee to review the list of orders from the website.
  2. The system should allow the employee to view the list of orders in different ways, By date and time or by price.
  3. The system should allow the employee to confirm an order.
  4. The system should allow the employee to contact the customer for confirmation and delivery.

Manager:

  1. The system allows the manager to view the list of employees in the restaurant with complete information about each employee.
  2. The system should allow the manager to send emails to the employees.
  3. The system should allow the manager to view a weekly made report to view the income of each day.

Feature:

  1. The system gives the customer the total control of his/her order.
  2. The system provides embedded maps that can easily locate the customer's address without having him/her to provide it manually.
